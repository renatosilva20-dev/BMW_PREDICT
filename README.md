#  BMW_PREDICT

📊Projeto pessoal de **Machine Learning** para análise e predição com dados de vendas globais da BMW com o objetivo de preditar quais carros irao vender mais, dividos em HIGH e LOW.

Este repositório contém um pipeline completo de Machine Learning em Python para carregar, pré‑processar, treinar e avaliar modelos preditivos com dados de vendas da BMW entre 2018 e 2025
---

##  Estrutura do Projeto

O repositório inclui os seguintes arquivos principais:

| Arquivo | Descrição |
|--------|------------|
| `bmw_global_sales_2018_2025.csv` | Dataset de vendas globais da BMW entre 2018–2025. 
| `load_data.py` | Script para carregar e visualizar os dados. 
| `preprocess.py` | Pré‑processamento dos dados antes de treinar o modelo. 
| `features_target.py` | Seleção de features e definição do target (variável de saída).
| `train_test.py` | Treinamento e avaliação de modelos de ML. 
| `plots.py` | Visualizações e gráficos para análise. 
| `main.py` | Script principal para rodar todo o pipeline

---

##  Tecnologias e Bibliotecas

O projeto foi desenvolvido em **Python 3.14** utilizando as seguintes bibliotecas:

- `pandas` – manipulação de dados
- `numpy` – cálculos numéricos
- `scikit‑learn` – modelos e métricas de ML
- `matplotlib` / `seaborn` – visualização de dados
- Outras (definidas no código conforme necessário)
